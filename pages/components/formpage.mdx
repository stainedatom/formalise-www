# `<FormPage/>`

The form page that is rendered sequentially based on the `<Button/>` component API:
- `children: ReactNode` - *required* the content of the form page. These are meant to be the input fields of your form.
- `style: CSSProperties` - the style to be passed on to the form page component.
- `className: string` - the class name to be passed on to the form page component.
