# About <span style={{backgroundColor: 'black', color: 'white'}}>{`<Formalise/>`}</span>

While attempting to build a complex form for a project of mine, I've come to realise how imperative of a task this can suddenly become. So I built Formalise, a form builder that abstracts away the imperative code so that you, the developer, can build paginated forms with ease.

I hope this piece of software will save you many hours to come.
